# CSC335 Project 6: Hack Assembler

A Python implementation of a two-pass assembler for the Hack computer platform (Nand2Tetris).

## Structure
- `assembler.py` – main program
- `test_cases/` – sample ASM test inputs
- `output/` – generated HACK binary outputs

## How to Run
1. Edit `assembler.py` as needed.
2. Place your `.asm` files in `test_cases/`.
3. Run the assembler and check outputs in `output/`.
